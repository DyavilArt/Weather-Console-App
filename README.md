# ☁️ Weather-Console-App

Консольное приложение для получения текущей погоды с использованием **Open-Meteo API** (бесплатно, без ключа).

## 🚀 Возможности

- Поиск погоды по названию города (с автоматическим определением координат)
- Отображение текущей температуры, давления и кода погоды
- Расшифровка кода погоды в понятное описание (ясно, дождь, снег и т.д.)
- Обработка ошибок (город не найден, проблемы с сетью)
- Циклический ввод для проверки нескольких городов за один запуск

## 🛠️ Используемые технологии

- **C# 13** / **.NET 11**
- **System.Text.Json** — для парсинга JSON
- **HttpClient** — для HTTP-запросов
- **Open-Meteo API** — источник погодных данных (бесплатный, без ключа)
- **Geocoding API** — преобразование названия города в координаты

## 📦 Установка и запуск

```bash
git clone https://github.com/ВАШ_НИК/Weather-Console-App.git
cd Weather-Console-App/Weather-Console-App
dotnet run
```
